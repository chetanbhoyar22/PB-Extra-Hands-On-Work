Q 1 : Given array of students marks [23, 45, 67, 78, 91] add 10 in all the elements of an array by using map and forEach function?
        Using Map -
                            let arr1 = [23, 45, 67, 78, 91]

                            let mks1 = arr1.map(num => num + 10);
                            console.log(mks1); //[33, 55, 77, 88, 101]
                            
        Using forEach -
                            arr1.forEach((value,index) => {
                                arr1[index] = value + 10;
                            })
                            console.log(arr1); //[33, 55, 77, 88, 101]

Q 8 : Explain call bind and apply in JS?
                Call apply bind is basically used to borrow function methods.

                1. Call - It borrows the function method and passes the object as an argument.

                var obj={
                    name: "Ajay",
                    age:24
                }
                function add(city){
                    console.log("Name is "+this.name+" age is "+this.age+" & city is "+city);
                }
                add.call(obj,"Mumbai");
                
                //Name is Ajay age is 24 & city is Mumbai

               2. Apply - It borrows the function method and passes the object as an argument. The only difference is it passes additional argument inside the array.

                var obj={
                    name: "Arun",
                    age:34
                }
                function add(city){
                    console.log("Name is "+this.name+" age is "+this.age+" & city is "+city);
                }

                add.apply(obj,["Delhi"]);

                //Name is Arun age is 34 & city is Delhi
        
                3. Bind - It borrows the function method and passes the object as an argument and returns a new function ie. copy of that function.
        
                var obj={
                    name: "Arav",
                    age:25
                }
                function add(city){
                    console.log("Name is "+this.name+" age is "+this.age+" & city is "+city);
                }
        
                var response = add.bind(obj,"Jaipur");
                response();

                //Name is Arav age is 25 & city is Jaipur
Q 9 : Explain Prototype Chaining?
                Each object has a private property which holds a link to another object called its prototype. That prototype object has a prototype has a prototype of its own and so 
                on until it reaches to null as its prototype. Hence this chain is called prototype chaining.

                // pocket -> bed -> table -> head

                let head = {
                    glasses: 1
                }

                let table = {
                    pen: 3,
                    __proto__: head
                }

                let bed = {
                    sheet: 1,
                    pillow: 2,
                    __proto__: table
                }

                let pockets = {
                    money: 2000,
                    __proto__: bed
                }
