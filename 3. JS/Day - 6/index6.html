Function -

Function is a piece of code which we will be used for reusability, it means write once & call multiple times or call anywhere.

Function Types -

I. Function without parameters, without return. 

                    function add(){
                        console.log('Hi')
                    }
                    add();  //Hi
II. Function with params, without return. 
                    function name(userName){
                        console.log(userName);
                    }
                    name('Chetan'); //Chetan
III. Function with params, with return. 
                    function multiply(a,b){
                        return a*b;
                    }
                    console.log(multiply(2,9)); //18

  • 1. Function Declaration

    • Function declaration defines a function with specified parameters. Here we have to give function name. Function declarations are hoisted. 

                        function wish (){
                         return('Hi!');
                    }
                    var result = wish();
                    console.log(result);  //Hi!
  • 2. Function Expression

    • Function Expression is similar to function declaration but here the function name is ommited. Function expression are not hoisted since during hoisting it treats it as a function variable. 

                        function add(a,b){
                        return a+b;
                    }
                    console.log(add(2,9)); //11
  • 3. Function Declaration VS Function Expression

    • Function Declarations its automatically hoisted but fun expressions not hoisted and treat as a variable so it shows undefined.

  • 4. Function Constructor

    • Function constructor is used to create objects. It use constructor function by using new keyword

                        var sub = new Function ("num1", "num2", "{return num1 - num2}");
                    console.log(sub (21,11)); //10
  • 5. Immediately Invoked Function Expression (IIFE)

    • IIFE is immediately invoked function expression which gets executed as soon as it is created. It is basically a anonymous function which can be executed only once. 

                        (function(){
                        console.log('IIFE');
                    })() //IIFE
  • 6. First Class Functions

    • First class function is function which can be passed as an argument inside a another function or it can be returned by a function or it can assigned as a value to another function. 

                        function sub(a,b){
                        return a-b;
                    }
                    
                    function subCall(sub){
                       return sub(15,5);
                    }
                    
                    console.log(subCall(sub)); //10
  • 7. Callback Functions

    • Callback function is a function which can be passed as an argument to another function. 

                        function hello (param){
                        param();
                        console.log('hello');
                    }
                    function hi(){
                        console.log('hi');
                    }
                    hello(hi);   //hi //hello
  • 8. Higher Order Functions

    • Higher order function is a function which can accept a function as a parameter or return a function. 

                        function submitHomeWork (teacher){
                        teacher();
                        console.log('submit home work');
                    }
                    function copyHomeWork(){
                        console.log('copy home work');
                    }
                    submitHomeWork(copyHomeWork); //copy home work //submit home work
  • 9. Closures

    • It is a combination of both inner and lexical environment created by the inner function. We are able to access the variable from outer scope. 

                        function outer(){
                        let a=20;
                        function inner(){
                            console.log(a);
                        }
                        inner();
                    }
                    outer();      //20
  • 10. How Functions Get Executed?

    • The code inside a function is not executed when the function is defined. The code inside a function is executed when the function is invoked. It is common to use the term "call a function" instead of "invoke a function". It is also common to say "call upon a function", "start a function", or "execute a function".
