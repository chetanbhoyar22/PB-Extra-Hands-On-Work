Arrays Methods - index, map, filter, reduce, push, pop, shift, unshift, some & every, slice

Array -

We can store any type of data type.

Index -

The first index of an array is stated with 0. 

                syntax - const value = arr.indexOf(2);

Map -

Map is an method in js, which is used to transform an array i.e., which will iterate through each and every element. 

                syntax - const result1 =arr.map(x => x*4);

Filter -

Filter which checks the condition and filter all the elements in array and return you a new array. 

                syntax - const result2 = arr.filter(x => x>2);

Reduce -

whenever the sum of all elements in an array or whenever we want to find the max value in array. 

                syntax - const total = arr.reduce((accumulator, current) => accumulator = accumulator + current, 0);

Push -

It will add one more value at the end of the array. 

                syntax - arr.push(8);

Pop -

It removes an element from the end of the array. 

                syntax - arr.pop();

Shift -

It will remove first element an array. 

                syntax: arr.shift();

Unshift -

It will add one value at the first of an array. 

                syntax - arr.unshift(1);

Some & Every -

Both these are take in a condition and check if all the elements in th array is satisfying the condition or not, they always return a boolean value. 

                some syntax - const val2 = arr.some(x => x%2 == 0); 
        
                every syntax - const val3 = arr.every(x => x%2 == 0);

Slice -

It will not manipulate the array, it will try to remove elements from an array. 

                syntax - const val4 = arr.slice(1,3); 
        
                1. Start Index : Element in this index will be included in the new array. 
        
                2. End Index: Element in this before index will be included.

Note - Example are executed in js file.
