In our Javascript we are having three types of variables -

1. var
var is a Global or Functional Scoped.
var can be reassigned.
var can be redeclare.
var can be modified later.

2. let
let is a Block Scoped.
let cannot be reassigned.
let cannot be redeclare.
let can be modified later.

3. const
const is a Block Scoped.
const cannot be reassigned.
const cannot be redeclare.
const cannot be modified later.


Hoisting - Hoisting is a default Javascript Mechanism where our variables and Functions declarations are moved to top of their Scope before code execution (),only declarations are hoisted not initializations.

Dead Zone - The time between the Variables hosted and the variables get initialized is called the Temporal Dead Zone
Let and const are also hoisted and are in Temporal Dead Zone
