Strings Methods : charAt, indexOf, split, trim, string reverse, join, slice, concatenation, replace, replaceAll

String -

It is a collection of characters.

String Length -

It's counting a number of characters in a string. 

                let myStr = "ChetanBhoyar";
                let len = myStr.length;
                console.log(len); //12

charAt -

If we want to find the characters of string with the help of indexes we will use charAt. 

                let chr = myStr.charAt(5);
                console.log(chr); //n

IndexOf -

It shows index position of string. 

                let io = myStr.indexOf("t");
                console.log(io); //3

Split -

converts string to an array, split method returns the new array, does not the change the original one. 

                let sp = myStr.split("");
                console.log(myStr);  //ChetanBhoyar
                console.log(sp); //(12) ['C', 'h', 'e', 't', 'a', 'n', 'B', 'h', 'o', 'y', 'a', 'r']

Reverse & Join -

It reverse the string & It converts array to string. 

                let rev1 = myStr.split("").reverse();
                console.log(rev1);  //(12) ['r', 'a', 'y', 'o', 'h', 'B', 'n', 'a', 't', 'e', 'h', 'C']
                let rev2 = myStr.split("").reverse().join("");
                console.log(rev2);  //rayohBnatehC

Trim -

It removes unnecessary spaces. 

                let str="             AjayDevgan";
                console.log(str.length);
                let tr = str.trim(); //23
                console.log(tr); //AjayDevgan
                console.log(tr.length); //10

Slice -

It is used to extract datac, it will not modify the original. 

                let stg1 = ["hi", "hello", "chetan", "how", "are", "you"]
                        //   0      1         2        3      4      5
                let slc1 = stg1.slice(1,3);
                console.log(slc1); //(2) ['hello', 'chetan']

Splice -

It is used for extraction and adding the elements and it will modify the original. 

                let stg2 = ["hi", "hello", "chetan", "how", "are", "you"]
                        //   0      1         2        3      4      5
                let slc4=stg2.splice(1,3);
                console.log(slc4);  //(3) ['hello', 'chetan', 'how']

Concatenation -

It concat two string. 

                var text1="Akshay";
                var text2="Kumar";
                var text3=text1.concat(" ",text2);
                console.log(text3); //Akshay Kumar

Replace -

It replace single occurrence. 

                newStr="I will go to delhi, I will eat chinese food."
                let rep1 = newStr.replace("will", "have");
                console.log(rep1); //I have go to delhi, I will eat chinese food.

ReplaceAll -

It replace all occurrences. 

                newStr="I will go to delhi, I will eat chinese food."
                let rep2 = newStr.replaceAll("will", "have");
                console.log(rep2); //I have go to delhi, I have eat chinese food.

Note - Example are executed in js file.
