CSS - Linear Gradient, Radial Gradient, float, clear, sizing - #border#content& z-index

linear-gradient -Perhaps the most common type of gradient we see in web design is the linear-gradient(). It’s called “linear” because the colors flow from left-to-right, top-to-bottom, or at any angle you chose in a single direction.

radial-gradient - A radial gradient differs from a linear gradient in that it starts at a single point and emanates outward. Gradients are often used to simulate a light source, which we know isn’t always straight. That makes the transitions between colors in a radial gradient seem even more natural.

float - The float property is used for positioning and formatting content

clear - The clear property specifies what should happen with the element that is next to a floating element.
Border-Box - The padding and border are included in the width and height.

Content-Box - The width & height of the element only include the content. In other words, the border, padding and margin aren’t part of the width or height. This is the default value
The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order
